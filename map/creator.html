General usage of this tools is.
  • Make a selection
  • Set tile
  • Edit note if any
  • Edit descriptive tekst

Make a selection

Click on a single cell to choose it. Control/command click on cells to add them to this selection. You can alse drag a lasso around cells to choose several. Currently it is not possible to deselct part of an lassoed area.

Set a tile

After a selection has been made navigate to the tiles on the bottom, as you hover them the cells in the grid will change. Mouse click on the desired tile to set it.

Mark a cell as not ocupiable

In a battlemap some cells are not meant to be ocupiable by monsters and miniatures. By default all cells are ocupiable. In order to make a cell not ocupiable click on the lock icon.

If you want to make the cell ocupiable again simply select it and click the unlock icon.

Cancel/Remove a selection

Press the X close icon to remove a selection and close the selection menu

Clear cells

When a selection has been made click on the trash icon to reset the selected cells to default. As requested by Kraake
